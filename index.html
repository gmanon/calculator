Welcome to Simple Javascript Calculator.

How to use it? To use it, just download the zip or gunzip, unpack your package and open the html page on your browser. Make sure the CSS and js folders are there.

The calculator can be converted into a module, it can be used in a pop-up window to help your user(students) with their calculations.

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
